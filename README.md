# vrcovid-parcours
UptaleDataAnalysis is a project that aims to provide in-depth individual and group behaviour analysis from [Uptale](https://www.uptale.io/en/home)'s data exports

## Features
 - Platform-agnostic
 - Client-sided
 - Compatible with any Uptale Project
